☕ BrewMap – Cafe Finder Web Application

BrewMap is a responsive web application that helps users discover nearby cafes using real-time geolocation. The application integrates Leaflet.js with OpenStreetMap (Overpass API) to provide an interactive map experience and accurate location-based search.

✨ Features
📍 Real-time geolocation
☕ Nearby cafe discovery
🗺️ Interactive maps with Leaflet.js
🔍 Search and location filtering
🌙 Dark mode support
📱 Fully responsive UI
⚡ Express.js backend for API handling
🛠️ Tech Stack
HTML5
CSS3
JavaScript
Node.js
Express.js
Leaflet.js
OpenStreetMap (Overpass API)
🎯 Purpose

This project was developed to provide a simple, fast, and user-friendly platform for locating nearby cafes while demonstrating skills in frontend development, backend integration, REST APIs, and location-based services.
